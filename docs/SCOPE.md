# Project Scope

## Funcionalidades incluidas (MVP)

La primera versión de la aplicación incluirá únicamente las siguientes funcionalidades:

- Registro manual de gastos
- Almacenamiento persistente de los gastos
- Consulta de gastos registrados
- Resumen básico de gastos por período y categoría

Estas funcionalidades representan el núcleo del problema que se desea resolver.

## Funcionalidades fuera de alcance (futuras mejoras)

Las siguientes funcionalidades quedan fuera del alcance de esta versión inicial:

- Gestión de presupuestos
- Visualización avanzada mediante gráficos
- Interfaz web o gráfica
- Exportación de datos a otros formatos
- Sincronización con servicios externos

Estas características podrían incorporarse en versiones futuras sin necesidad de rediseñar la aplicación.
