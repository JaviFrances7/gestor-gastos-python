# User Flow

## Inicio de la aplicación

El usuario ejecuta la aplicación desde la terminal y accede a un menú principal con las acciones disponibles.

## Registro de un gasto

1. El usuario selecciona la opción para añadir un gasto.
2. Introduce la información requerida (importe, categoría, fecha, descripción).
3. El sistema valida los datos.
4. El gasto se guarda correctamente.

## Consulta de gastos

1. El usuario selecciona la opción de consulta.
2. El sistema muestra los gastos registrados según el criterio seleccionado.
3. El usuario obtiene un resumen claro de su información financiera.

## Cierre

El usuario puede salir de la aplicación en cualquier momento desde el menú principal.
